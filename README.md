# 知识库管理方案

## 分级结构

### 🔥 Hot (热点)
- 路径: `knowledge/hot/`
- 内容: 近7天活跃、高频访问、当前项目相关
- 索引: 全文索引，快速检索
- 保留: 30天未访问自动降级到Warm

### 🌡️ Warm (温盘)
- 路径: `knowledge/warm/`
- 内容: 1-3个月内研究过、有参考价值
- 索引: 标题+摘要索引
- 保留: 90天未访问自动降级到Cold

### ❄️ Cold (冷盘)
- 路径: `knowledge/cold/`
- 内容: 过期资料、错误方向、已替代技术
- 索引: 仅文件名索引
- 保留: 长期保存，但不参与常规检索

## 降级规则

1. **Hot → Warm**: 30天未访问
2. **Warm → Cold**: 90天未访问
3. **手动降级**: 技术过时、错误方向、新方案替代

## 文件命名规范

```
YYYY-MM-DD_[类型]_[关键词].md

类型:
- tech: 技术动态
- tool: 工具发现
- paper: 论文研究
- skill: Skill开发
- method: 方法论

示例:
2026-03-21_tech_MSA-memory-attention.md
2026-03-21_tool_screenpipe.md
```
