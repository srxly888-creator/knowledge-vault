零成本构建AI专业团队，147个Markdown Agent，覆盖12个部门

- **GitHub Stars**: 46.8k

## 技术特点

### 1. 公司组织"代码化"
- 147个专业Agent
- 覆盖工程、产品、设计、增长等12个部门
- 每个Agent = 一个高质量Markdown文件

### 2. 标准化定义
- 角色个性
- 工作流程
- 交付物模板
- 评估标准

### 3. 多工具支持
- Claude Code
- Cursor
- Aider
- Gemini CLI
- Windsurf
- OpenClaw

## 使用方式

### 单Agent调用
把对应的.md文件丢进上下文，AI切入专家视角

### 多Agent协作
- 手动链式调用
- Agents Orchestrator协调工作流

## 典型闭环示例

```
产品经理Agent → 梳理需求分析与PRD
↓
后端架构师Agent → 承接PRD，输出技术方案与架构图
↓
前端工程师Agent → 根据架构图，生成完整业务代码
↓
增长专家Agent → 制定推广策略与文案
```
