# 杰哥的十个顶级Claude Code Skills

**来源**: https://x.com/0xjason/status/2033909937823465887
**发现日期**: 2026-03-21
**优先级**: ⭐⭐⭐ 高

## 核心定位

Claude Code不是普通代码补全，而是能直接干活的编程代理

## 十大Skills

### 1. Superpowers ⭐⭐⭐
- 打包20+可组合Skill
- 覆盖软件开发完整流程
- brainstorming + TDD最常用
- brainstorming：先问问题再动手
- TDD：先写测试再写实现

### 2. Planning with Files ⭐⭐⭐
- 解决Plan Mode上下文丢失问题
- 规划/进度/知识写进Markdown文件
- 文件在磁盘上不会丢

### 3. UI UX Pro Max ⭐⭐
- 67种UI风格
- 161套行业配色
- 根据项目自动推荐设计系统
- 支持React/Vue/Svelte/SwiftUI/Flutter

### 4. Code Review ⭐⭐⭐
- 多Agent并行审查
- 逻辑/安全/风格分别检查
- 置信度过滤，减少假阳性
- 注意：大PR token消耗猛

### 5. Code Simplifier ⭐⭐
- 检查重复逻辑
- 合并条件分支
- 不改功能，只做简化

### 6. Playwright测试自动化
- 自动写Playwright脚本
- 启动浏览器、跑测试、截屏
- 有问题自动调试

### 7. Ralph Loop ⭐⭐
- Stop Hook拦截退出
- 检查完成条件
- 没满足就塞回去继续

## 安装命令

```bash
claude plugin marketplace add <repo>
claude plugin install <name>
```

## 使用建议

- 别全开，只启用常用的
- brainstorming + TDD两个就够
- 大PR先拆再审
