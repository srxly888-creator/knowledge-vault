# Open SWE - 企业内部AI编程助手框架

**来源**: https://x.com/github_daily/status/2034842415941788047
**GitHub**: https://github.com/langchain-ai/open-swe
**发现日期**: 2026-03-21
**优先级**: ⭐⭐⭐ 高

## 核心定位

LangChain开源基础框架，帮助团队构建企业内部AI编程助手（不用从零手搓）

## 技术栈
- Deep Agents
- LangGraph

## 关键能力
- 沙箱隔离执行
- 精选工具集（约15个）
- 子智能体编排
- 原生集成：Slack / Linear / GitHub
- 通过 AGENTS.md + 任务上下文注入背景
- 中间件实现确定性流程控制

## 架构特点
组合式架构（不是fork代码），可持续继承Deep Agents演进。
