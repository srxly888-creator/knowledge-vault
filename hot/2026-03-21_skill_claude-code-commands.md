# Claude Code 隐藏命令

**来源**: https://x.com/khazix0918/status/2035006758226207221
**发现日期**: 2026-03-21
**优先级**: ⭐⭐⭐ 高

## 核心命令

### 1. `/btw` - 异步插话
- **功能**: 执行中插问，不污染上下文
- **场景**: 长任务中突然想到问题
- **优势**: 不中断主任务，复用prompt缓存

### 2. `/rewind` - 时空穿梭
- **触发**: 双击Esc
- **功能**: 代码/对话独立回滚
- **场景**: 试错、实验性开发
- **选项**: 
  - 回退代码和对话
  - 回退对话保留代码
  - 回退代码保留对话
  - 压缩释放上下文

### 3. `/insights` - 使用报告
- **功能**: 生成HTML报告分析使用习惯
- **内容**: 常用命令、重复模式、优化建议
- **建议**: 每月运行一次

### 4. `/model opusplan` - 隐藏模式 ⭐
- **功能**: Opus规划 + Sonnet执行
- **优势**: 省token，Pro用户福音
- **场景**: 复杂任务规划
- **注意**: 列表不可见，真正的隐藏命令

### 5. `/simplify` - 三Agent联审
- **功能**: 并行启动3个Agent审查代码
- **角度**: 代码复用、代码质量、运行效率
- **场景**: 大功能更新后review

### 6. `/branch` - 平行宇宙
- **原名**: /fork
- **功能**: 会话分叉
- **场景**: 同一思路试不同实现

### 7. `/loop` - 定时循环
- **用法**: `/loop 5m 检查部署状态`
- **过期**: 3天自动删除
- **长期**: 需使用面板

### 8. `/rc` - 远程控制
- **功能**: 生成URL，手机访问Claude Code
- **特点**: 完全同步，代码仍在本地
- **支持**: Telegram, Discord

### 9. `/export` - 导出对话
- **功能**: 导出Markdown
- **场景**: 存档、跨Agent协作

## 快捷键

- `Ctrl+V` - 粘贴截图（Mac也用Ctrl）
- `Ctrl+J` - 换行
- `Ctrl+R` - 搜索历史prompt
- `Ctrl+U` - 删除整行

## 最佳实践

1. 长任务多用`/btw`避免上下文污染
2. 实验性开发用`/rewind`试错
3. Pro用户必开`/model opusplan`
4. 每次大更新后`/simplify`审查
5. 重要对话及时`/export`备份
