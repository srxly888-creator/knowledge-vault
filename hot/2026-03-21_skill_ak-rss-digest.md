## 核心价值

Andrej Karpathy推荐的90个高质量长文章RSS订阅源

## 特点

### 1. 源质量保证
- 90个订阅源
- 全部经过AK把关
- 更新频率不高，但质量高

### 2. 智能评分
- 每次新更新自动打分（1-10分）
- 只输出7分以上的文章
- 附带推荐语

### 3. 输出格式
- 文章地址
- 推荐语

## 安装

```bash
npx skills add rookie-ricardo/erduo-skills --skill ak-rss-digest
```

## 适用场景

- 追踪高质量技术文章
- 减少信息噪音
- 自动筛选优质内容
