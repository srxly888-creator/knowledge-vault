## 🏗️ Skills 架构

### 标准 Skill 结构

```
skill-name/
├── SKILL.md          # 技能说明
├── tools/            # 工具定义
├── prompts/          # 提示模板
└── examples/         # 示例
```

### Skills vs MCP 对比

| 特性 | Skills | MCP |
|------|--------|-----|
| 格式 | Markdown + JSON | Protocol |
| 动态性 | 静态定义 | 动态调用 |
| 复杂度 | 低 | 高 |
| 适用场景 | 简单任务 | 复杂集成 |
