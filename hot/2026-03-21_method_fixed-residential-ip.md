# 固定住宅IP配置教程

**来源**: https://x.com/jiroucaigou/status/2034860543543321044
**发现日期**: 2026-03-21
**优先级**: ⭐⭐ 中

## 核心问题

机场IP不纯净、不固定：
- 一个IP几万人用
- 地理位置频繁跳变
- 易触发封号

## 解决方案

**代理链路**: 机场节点 → 静态住宅IP → 目标网站

## 准备材料

1. Clash Verge（客户端）
2. 机场订阅（最便宜即可）
3. 静态住宅IP（购买）
   - cliproxy（便宜）
   - ipfly（贵但稳定）

## Windows配置

### 1. 导入订阅
- Clash Verge填入机场链接

### 2. 配置静态节点
```
格式: socks5://用户名:密码@服务器地址:端口#备注名
```

### 3. 构建代理链
- 右键订阅 → 编辑代理组
- 类型: Relay
- 顺序: 机场节点 → 静态IP（不可颠倒）

### 4. 启用
- 模式: Global
- 开启系统代理 + TUN模式

### 5. 验证
- 访问 ping0.cc
- 确认IP为静态住宅IP

## Mac/iOS配置

### 小火箭配置
1. 添加节点 → 类型SOCKS5
2. 填写静态IP信息
3. 代理通过 → 选择机场节点
4. 完成

## Android配置

### YAML配置文件
```yaml
proxies:
  - 机场节点信息
  - 静态IP信息

proxy-groups:
  - name: "链式代理"
    type: relay
    proxies:
      - 机场节点
      - 静态IP
```

### 导入使用
1. 配置文件存到手机
2. Clash导入YAML
3. 选择链式代理
4. 测试连通性

## 应用场景

- 推特防封
- Claude防封
- TikTok防封
- 撸毛防关联

## 注意事项

- 顺序不可颠倒：机场 → 静态IP
- 验证IP纯净度
- 指纹浏览器更安全（撸毛）

## 成本估算

- 机场: $5-20/月
- 静态住宅IP: $10-50/月
- 总计: $15-70/月
