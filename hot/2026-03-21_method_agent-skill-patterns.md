# Agent Skill 设计模式

**来源**: https://x.com/kengguanglong/status/2034491445646909606
**发现日期**: 2026-03-21
**优先级**: ⭐⭐⭐ 高

## 核心观点

格式不再是问题（30+ Agent工具统一标准），真正难点是**Skill逻辑组织**

## 五种设计模式

### 1. 工具封装
- 封装底层API/服务
- 统一接口调用
- 隔离复杂性

### 2. 生成器
- 自动生成内容/代码
- 基于模板+参数
- 批量产出

### 3. 审查器
- 质量检查
- 规则验证
- 输出过滤

### 4. 反转模式 ⭐
- **核心**: Agent先采访用户，问清需求再动手
- **避免**: 瞎猜用户意图
- **场景**: 复杂任务、模糊需求

### 5. 流水线
- 多步骤串联
- 前一步输出→后一步输入
- 标准化流程

## 组合应用

模式可互相组合：
- 生成器 + 审查器 = 质量保证
- 反转模式 + 流水线 = 需求澄清→执行
- 工具封装 + 生成器 = 自动化服务

## 实战价值

- 可直接落地
- 谷歌官方实战经验
- ADK开发必备
