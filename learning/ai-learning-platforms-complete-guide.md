## 2. Coursera AI

### 2.1 平台特色

#### 核心优势
- **顶尖大学合作**: Stanford、MIT、DeepMind、IBM、Google
- **学术权威性**: 课程内容由大学教授和行业专家共同设计
- **灵活学习模式**: Audit（免费旁听）、付费证书、学位项目
- **全球认可度**: 证书被 1000+ 家企业认可
- **多语言支持**: 英文、中文、西班牙文等

#### 课程类型
1. **单个课程** - 4-12 周，灵活入门
2. **专项课程** - 3-7 门课程，系统学习
3. **职业证书** - 就业导向，含项目
4. **在线学位** - 硕士/学士，完全在线

### 2.2 顶级 AI 课程推荐

#### 🏆 Andrew Ng 经典三件套

| 课程 | 周期 | 难度 | 项目数 | 评价 |
|------|------|------|--------|------|
| **Machine Learning** | 11 周 | ⭐⭐⭐ | 5 | 4.9/5 ⭐ |
| **Deep Learning Specialization** | 5 个月 | ⭐⭐⭐⭐ | 8 | 4.8/5 ⭐ |
| **AI for Everyone** | 4 周 | ⭐⭐ | 0 | 4.8/5 ⭐ |

**Machine Learning（2011, 重制版 2022）**
- **讲师**: Andrew Ng（斯坦福教授、Coursera 联合创始人）
- **内容**: 线性回归、逻辑回归、神经网络、SVM、K-means、PCA
- **特色**: 
  - 经典算法深入讲解
  - 理论与实践并重
  - Octave/Matlab 编程
  - 超过 500 万学员
- **适合**: 零基础入门、理论扎实者
- **项目**:
  1. 线性回归预测房价
  2. 逻辑回归识别手写数字
  3. 神经网络识别图像
  4. 异常检测
  5. 推荐系统

**Deep Learning Specialization（5 门课程）**
- **课程 1**: Neural Networks and Deep Learning
- **课程 2**: Improving Deep Neural Networks（优化、正则化）
- **课程 3**: Structuring Machine Learning Projects（ML 策略）
- **课程 4**: Convolutional Neural Networks（计算机视觉）
- **课程 5**: Sequence Models（NLP、RNN、Transformer）
- **特色**:
  - 从基础到前沿（Transformer、BERT）
  - 8 个实战项目
  - 使用 TensorFlow/Keras
- **项目**:
  1. 识别猫
  2. 深度神经网络优化
  3. Bird recognition（图像分类）
  4. 人脸识别
  5. Neural Style Transfer
  6. Jazz improviser（LSTM）
  7. 触发词检测
  8. Transformer 翻译

**AI for Everyone**
- **受众**: 非技术人员、管理者
- **内容**: AI 概念、应用、伦理
- **特色**: 无需编程，4 小时快速了解 AI

### 2.4 学习体验

#### 优点 ✅
- 内容质量极高（学术+行业双重保证）
- 证书认可度高
- 社区活跃（论坛、Peer Review）
- 移动端体验好
- 离线下载支持
- 作业和项目有即时反馈

#### 缺点 ⚠️
- 价格较高（证书模式）
- 部分课程更新较慢
- 实战项目相对 Udacity 较少
- 缺少导师一对一指导
- 中文支持有限（除少数课程）

### 10.4 证书 vs 真实能力

**真正重要的是**:
1. **项目实战**:
   - 可以展示的 GitHub 项目
   - 可以讲述的项目经历
   - 解决实际问题的能力

2. **技术深度**:
   - 理解原理，不只是调包
   - 能解释模型为什么工作
   - 能优化模型性能

3. **工程能力**:
   - 能部署模型
   - 能处理数据
   - 能写出生产代码
