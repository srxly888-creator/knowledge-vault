## 📚 官方文档结构

### 一、Getting Started（入门）

#### 1. Overview（概览）
- **官网**: https://docs.anthropic.com/en/docs/overview
- **定位**: Claude Code 是一个 AI 编程助手，理解整个代码库，跨多文件工作
- **核心能力**:
  - 自动化重复性工作
  - 构建功能和修复 bug
  - 创建 commits 和 PR
  - 连接工具（MCP）
  - 自定义指令、技能、钩子
  - 运行代理团队
  - 脚本和自动化

#### 2. Quickstart（快速开始）
- **官网**: https://docs.anthropic.com/en/docs/quickstart
- **安装方式**:
  ```bash
  # macOS/Linux/WSL
  curl -fsSL https://claude.ai/install.sh | bash
  
  # Windows PowerShell
  irm https://claude.ai/install.ps1 | iex
  
  # Windows CMD
  curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
  ```

- **启动**:
  ```bash
  cd your-project
  claude
  ```

- **首次使用**: 会提示登录
- **进阶设置**: https://docs.anthropic.com/en/docs/setup

#### 3. Changelog（更新日志）
- **官网**: https://docs.anthropic.com/en/docs/changelog

## 💡 实用技巧

### 1. CLAUDE.md 文件
- **位置**: 项目根目录
- **作用**: 存储项目特定的指令和上下文
- **示例**:
  ```markdown
  # 项目说明
  
  ## 技术栈
  - Node.js 18+
  - TypeScript
  - React
  
  ## 编码规范
  - 使用 ESLint
  - 单元测试覆盖率 > 80%
  ```

### 2. 自动记忆
- Claude Code 会自动记住重要的上下文
- 存储在 `~/.claude/memory/` 目录

### 3. 多文件操作
- Claude Code 理解整个代码库
- 可以跨多个文件进行重构

### 4. Git 集成
- 自动创建 commits
- 自动生成 PR 描述
- 代码审查建议

## 🚀 快速参考

### 常用命令
```bash
# 启动 Claude Code
claude

# 指定工作目录
claude /path/to/project

# 使用特定模型
claude --model claude-3-5-sonnet-20241022

# 非交互模式
claude --print "your task"

# 版本信息
claude --version
```

### 环境变量
```bash
# API Key
ANTHROPIC_API_KEY=your_key

# 默认模型
CLAUDE_MODEL=claude-3-5-sonnet-20241022

# 调试模式
CLAUDE_DEBUG=true
```

## ⚠️ 注意事项

### 1. 费用
- 需要 Claude 订阅或 Anthropic Console 账户
- Terminal CLI 和 VS Code 支持第三方提供商

### 2. 数据安全
- 代码会上传到 Anthropic 服务器
- 注意敏感信息保护

### 3. 性能
- 大型代码库可能需要更多 token
- 使用 `claude-token-optimizer` 优化
