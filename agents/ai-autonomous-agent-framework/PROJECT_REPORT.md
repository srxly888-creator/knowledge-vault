## 核心特性实现

### ✅ 1. 自主任务分解

- **TaskDecomposer**：将复杂目标分解为可执行的子任务树
- 支持规则基分解（快速但不够智能）
- 支持 LLM 驱动分解（智能但成本较高）
- 任务依赖关系管理

### ✅ 2. 自我规划

- **PlanningEngine**：生成最优的任务执行计划
- 启发式规划：依赖图 + 拓扑排序
- 智能规划：使用 LLM 生成计划
- 约束条件支持（时间、成本等）

### ✅ 3. 自我执行

- **Executor**：执行具体任务并返回结果
- 工具注册和调用系统
- 错误处理和结果管理
- 支持规则基和 LLM 驱动执行

### ✅ 4. 自我评估

- **ReflectionModule**：评估执行结果并生成修正建议
- 质量评分系统（0-10分）
- 问题识别和建议生成
- 反馈机制

### ✅ 5. 记忆管理

- **MemorySystem**：管理短期、长期和工作记忆
- 记忆存储、检索和搜索
- 记忆导出和导入
- 记忆索引优化

## 使用方法

### 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行测试
python tests/test_basic.py

# 运行示例
python example.py

# 运行所有实验
./run_all.sh
```

### 基本使用

```python
from core import AutonomousAgent

# 创建 Agent
agent = AutonomousAgent(verbose=True)

# 注册工具
agent.executor.register_tool("web_search", your_tool)

# 运行 Agent
result = agent.run(
    goal="你的目标",
    tools={},
    constraints={"max_time": 10, "max_cost": 100}
)

print(f"结果: {result['result']}")
```
